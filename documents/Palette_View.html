Palette View

Visual designer 에서 화면을 만들기 위해 필요한 컴포넌트들은 palette view 에 존재합니다. Palette view 에서 컴포넌트를 선택하고 scene 의 원하는 위치에 클릭하면 컴포넌트가 배치됩니다.
또한 Palette view 의 컴포넌트를 마우스로 더블 클릭하거나 Ctrl 키를 누른 상태에서 더블 클릭하면 visual designer 의 현재 선택 위치에 컴포넌트가 배치됩니다.



1: Hide Palette
Palette view 를 감추고 visual designer 의 디자인 영역을 늘일 수 있습니다.

2: Pin Open
Palette category 를 열려있는 상태로 유지합니다.

3: Select
Visual designer 에 배치된 컴포넌트를 선택합니다.

4: 이동 가능 영역
Palette view 를 우측으로 이동하고 싶은 경우 이 영역을 마우스로 드래그하면 됩니다.

Context menu

Palette 의 컴포넌트를 마우스 오른쪽 버튼으로 클릭하면 다음과 같은 컨텍스트 메뉴가 나타납니다.



1: Insert Tag
선택된 Palette Component 를 visual designer 의 현재 caret 위치 에 배치합니다. Palette Component를 마우스로 더블 클릭해도 됩니다.

2: Wrap Tag
선택된 Component들을 감싸주는 부모 Component 를 생성해서 배치합니다. Palette Component 를 Ctrl 키를 누른 상태에서 마우스로 더블 클릭해도 됩니다.

3: Columns
Palette Component 아이콘과 설명을 격자 형태로 정렬합니다.

4: List
Palette Component 아이콘들과 설명들을 list 형태로 정렬합니다.

5: Icons Only
Palette Component 아이콘들을 설명없이 격자 형태로 정렬합니다.

6: Details
Palette Component 아이콘들에 대한 자세한 설명을 같이 보여줍니다.

7: Use Large Icons
Palette Compoent 아이콘으로 크기가 큰 아이콘 이미지를 사용합니다. 일부 Component는 큰 이미지 아이콘이 지원되지 않을 수 있습니다.

8: Setting
Palette view 에 대한 세부 설정을 변경 할 수 있습니다.
