Variables View

Variables view 에서는 debug view 에서 현재 선택된 stack frame 에 대한 variable 정보를 보여줍니다.



1: Name
선택된 variable 의 이름을 나타냅니다.

2: Value
선택된 variable 의 값을 나타냅니다.

3: Detail Pane
선택된 variable 을 텍스트로 표시해줍니다. Variable type 에 따라 텍스트 출력이 지원되지 않을 수 있습니다.

4: Show Type Names
Variable 의 type 을 함께 표시해줍니다. Layout > Show Columns 가 선택되지 않은 경우에만 사용가능합니다.

5: Show Logical Structures
현재 버전에서는 동작하지 않습니다.

6: Collapse All
펼쳐진 트리를 모두 접습니다.

7
다음과 같은 추가 메뉴를 사용할 수 있습니다.



8: Layout > Vertical
Detail pane 을 view 의 하단에 배치합니다.

9: Layout > Horizontal
Detail pane 을 view 의 오른쪽에 배치합니다.

10: Layout > Automatic
View 의 크기에 따라 detail pane 의 위치를 자동으로 배치합니다.

11: Layout > Variables View Only
Detail pane 을 감춥니다.

12: Show Columns
선택 해제하면 column 없이 variable 값을 텍스트로 보여줍니다.

13: Select Columns...
보여질 Column 을 선택합니다.



14: Show proto variables
선택하면 Prototype variable 을 보여줍니다. 현재 버전에서는 동작하지 않습니다.

15: Show 'this' variable
선택하면 This 객체의 variable 을 보여줍니다. 현재 버전에서는 동작하지 않습니다.

16: Show function variables
선택하면 function 객체의 variable 을 보여줍니다. 현재 버전에서는 동작하지 않습니다.
