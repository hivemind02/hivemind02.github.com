Console View

Console view 는 debug 모드로 application 이 실행되는 동안 출력되는 로그와 에러메시지등을 확인할 수 있는 view 입니다. Debug 모드 일때만 console view 에 메시지가 출력되며 일반 모드로 실행되면 출력되지 않습니다. 일반 모드 실행시 로그 메시지를 확인하기 위해서는 Emulator 에서 개발자용 콘솔(Developer Tools)을 사용해야 합니다.



1: Log Viewer
로그 메시지가 출력되는 화면입니다.

2: Clear Console
현재 화면에 출력된 메시지를 모두 지웁니다.

3: Scroll Lock
메시지가 출력될 때 화면이 자동 스크롤 되지 않도록 설정합니다.

4: Pin Console
Console 이 여러개 존재할 때 현재 console 이 항상 위로 유지되도록 합니다.

5: Display Selected Console
Console 이 여러개 존재할 때 console 들 간을 전환합니다.

6: Open Console
새로운 console 을 추가합니다.
