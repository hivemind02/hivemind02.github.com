JavaScript Source File Wizard

Javascript source file wizard 를 통해 js 파일을 추가할 수 있습니다.

Enter or select the parent folder
Js file 이 생성될 folder 의 경로를 입력하거나 project 에서 선택할 수 있습니다.

File name
생성될 js file 의 이름을 입력합니다. 확장자를 붙이지 않아도 생성되는 파일에는 확장자로 js 가 붙습니다.
