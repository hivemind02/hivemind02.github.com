네비게이션 구현하기

사용자에게 제공할 정보가 계층적으로 구성이되어 있을 때 일반적으로 Navigation 과 같은 방법을 통해 화면을 구성합니다. Mobello UI Framework에서는 이를 Scene Navigation 이라고 부르며 이를 쉽게 구현할 수 있도록 SequnceNavigator를 제공하고 있습니다.

이 문서는 Mobello UI Framework에서 제공하는 SequenceNavigation을 이용하여 화면간의 Navigation 방법에 대해 설명합니다.

App을 생성하는 기본적인 과정은 처음 시작하기 문서를 참고하십시오.

*이 데모는 WebKit 기반의 브라우저(Chrome, Safari)에서만 동작합니다.

개요

계층적으로 구성된 각 나라별 자동차 제조사 이름을 사용자에게 보여주는 예제로써 각 단계별 개발과정을 설명합니다.

예제에 사용하는 데이터는 JSON형태로 기술하고 다음과 같은 형태를 가집니다. 

{title: 'Nested List', items: [                                      
  {title: 'Asia', items: [                                           
    {title: 'Japan', items: [{title: 'Acura'}, {title: 'Honda'}, ...
    {title: 'Korea', items: [{title: 'Hyundai'}, {title: 'Kia'}]}    
  ]},                                                                
  {title: 'United Kingdom', items: [{title: 'Aston Martin'}, ...
  {title: 'Europe', items: [                                         
    {title: 'Germany', items: [{title: 'Audi'}, {title: 'BMW'}, ...
    {title: 'France', items: [{title: 'Citroen'}, {title: 'Renault'}, ...
  {title: 'United Staes', items: [{title: 'Buick'}, ...
]}

Step1: Main Controller 클래스 생성

Main Controller 클래스는 App이 구동될 때 Runtime에 의해 처음 실행되는 Class로써 이름을 'tau.sample.list.NestedList' 로 클래스를 생성하겠습니다.
계층적인 정보를 보여주기 위해 Mobello Framework에서 기본적으로 제공하는 'tau.ui.SequenceNavigator'를 상속받아 'tau.sample.list.NestedList'클래스를 정의합니다. SequenceNavigator 는 계층적으로 구성된 정보를 편리하게 Navigation 할 수 있도록 제작된 Controller 클래스 입니다.

tau.sample.list.NestedList
$class('tau.sample.list.NestedList').extend(tau.ui.SequenceNavigator).define({
  init: function () {
    var config = tau.getCurrentContext().getConfig();
    this.setRootController(new tau.sample.list.ListController(config.data));
  }
});

  • 최초로 실행되는 init() 메소드에서 사용할 예제에 필요한 데이터를 로딩

  • SeqeunceNavigator의 RootController로 'tau.sample.list.ListController' 클래스를 인스턴스를 설정

Step2: ListController 클래스 생성

주어진 데이터를 바탕으로 리스트 화면을 생성하는 Controller 클래스로써 'tau.ui.TableSceneController'클래스를 상속받아 정의합니다. 'tau.ui.TableSceneController'는 리스트형태의 화면을 쉽게 작성할 수 있도록 Mobello Framework에서 제공하는 Controller 클래스입니다.

tau.sample.list.ListController
$class('tau.sample.list.ListController').extend(tau.ui.TableSceneController).define({
  ListController: function (model) {
    this.model = model;
  },

  loadModel: function (start, size) {
    var table = this.getTable();
    table.addNumOfCells(this.model.items.length);
  },

  makeTableCell: function (index, offset) {
    var cell = new tau.ui.TableCell();
    cell.setTitle(this.model.items[offset + index].title);
    return cell;
  },

  cellSelected: function (current, before) {
    var table = this.getTable();
    if (current instanceof tau.ui.TableCell) {
      var path = table.indexOf(current).pop(); // index is array
      var item = this.model.items[path];
      if (item.items) {
        var detailController = new tau.sample.list.ListController(item);
        this.getParent().pushController(detailController); 
      }
    }
  },
  
  destroy: function () {
    this.model = null;
  }
});

  • 생성자에서 사용할 데이터는 인스턴스 변수로 설정

  • loadModel() callback메소드에서 Table컴포넌트로 로딩할 TableCell의 개수를 설정

  • makeTableCell() callback메소드에서 명시된 인덱스에 해당하는 TableCell의 객체를 생성하여 반환

  • cellSelected() callback메소드(사용자가 TableCell을 터치했을 때)에서 다음으로 보여줄 Controller를 생성하고 pushController()메소드를 통해 Navigation을 실행(parent는 SequenceNavigator 임)

  • destory() callback메소드에서 인스턴스변수인 model을 nullify시킴
