Button 사용하기

일반적으로 많이 사용하는 버튼 컴포넌트에 대해서 알아보고, Button 컴포넌트에서 제공하는 스타일을 적용해 보도록 하겠습니다.

목차:

  • Button이란?

  • 테마 적용하기

  • 상태별 버튼 스타일 적용하기

Button이란?

HTML element의 하나인 button을 모바일 버전에 맞게 만든 컴포넌트입니다. 일반적으로 탭했을 때 이벤트를 받아 사용자의 업무로직을 처리할 수 있습니다.

Mobello에서는 버튼 컴포넌트에 대한 다양한 스타일을 제공하고, 버튼의 상태에 따라 스타일을 적용할 수 있는 기능을 제공하고 있습니다.

테마 적용하기

Mobello에서 적용되어 있는 Button의 배경은 CSS3의 gradient가 적용이 되어 있습니다. 또한 버튼은 터치했을 때 탭했을 때 버튼을 터치하지 못하였을 때 다른 배경색상을 제공하고 있습니다.

사용자가 직접 CSS를 적용해도 무방하지만, 꽤 번거로운 작업이기 때문에 Mobello에서 스타일 클래스로 정의해서 바로 적용할 수 있습니다.

Mobello의 default테마는 버튼에 대해서 기본적으로 6개의 타입의 스타일 클래스를 제공하고 있습니다.

  • normal: gradient가 없고, shadow가 적용되지 않은 간단한 형태.

  • default : 회색계열

  • dark : 검정색 개열. 네비게이션바의 이전 버튼에 적용되어 있음.

  • red : 적색계열

  • khaki : 카키색 계열

  • green : 녹색계열

소스

데모: 현재 webkit 기반의 브라우저(chrome, safari)에서만 확인가능합니다.

API docs 

var normal = new tau.ui.Button({
  label: 'normal',
  styleClass: {
    type: 'normal'
  }
});
var def = new tau.ui.Button({
  label: 'default',
  styleClass: {
    type: 'default'
  }
});

상태별 버튼 스타일 적용하기

버튼 컴포넌트는 버튼의 상태에 따라 스타일과 라벨 텍스트를 적용할 수 있습니다.

여기서 버튼 상태는 다음과 같습니다.

  • normal : 일반적인 상태

  • disabled: 터치 불가능한 상태

  • highlighted: 터치 했을 때

  • selected: 탭하거나, 탭을 유지하고 있는 상태

설정할 수 있는 스타일은 다음과 같습니다.

  • 라벨 텍스트 값

  • 배경색

  • 배경이미지

  • 텍스트색

소스

데모: 현재 webkit 기반의 브라우저(chrome, safari)에서만 확인가능합니다.

API docs

일반적으로 사용자가 터치이벤트를 받아서 콜백함수에서 스타일을 적용할 수 있습니다. 

var button = new tau.ui.Button({
  id: 'btn1',
  label: 'normal'
});
button.onEvent(tau.rt.Event.TOUCHSTART, this.handleTouchStart, this);
button.onEvent(tau.rt.Event.TAP, this.handleTouchStart, this);
handleTouchStart: function (e, payload) {
  var src = e.getSource();
  src.setLabel('touchStart');
},
 
handleTap: function (e, payload) {
  var src = e.getSource();
  src.setLabel('tap');
 
  window.setTimeout(function () { src.setLabel('normal');}, 500);
},

다음은 텍스트 색상을 상태에 따라 설정한 예제입니다.

소스

데모: 현재 webkit 기반의 브라우저(chrome, safari)에서만 확인가능합니다.

API docs 

var button = new tau.ui.Button({
  textColor: {
   normal: 'Black',
   disabled: 'Gray',
   highlighted: 'Blue',
   selected: 'DarkBlue'
  }
});

button.onEvent(tau.rt.Event.TAP, this.handleTap, this);

현재 버전(v0.9.0M) Mobello에서는 상태별 스타일이 제대로 동작하지 않습니다.

버그사항

  • 터치했을 때 tau.ui.Button#setLabel, tau.ui.Button#setTextColor, tau.ui.Button#setBackgroundColor, tau.ui.Button#setBackgroundImage 에 설정한 highlighted 값이 적용되지 않습니다.

  • 탭 되었을 때 tau.ui.Button#setLabel , tau.ui.Button#setTextColor, tau.ui.Button#setBackgroundColor, tau.ui.Button#setBackgrounImage 에 설정한 selected 값이 적용되지 않습니다.

이후 버전(Jul 03, 2012)에서는 해당 버그가 수정되었습니다. 참고하시기 바랍니다.
