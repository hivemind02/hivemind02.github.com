theme mixins

Mobello에서는 전역으로 자주 사용하는 스타일룰에 대해 mixin로 정의하였습니다. 또한 컴포넌트내에서도 자주 사용하는 스타일룰에 대해서도 mixin로 정의하였습니다.

목차:

  • global mixins

  • 컴포넌트 mixins

    • Button

    • ScrollPanel

    • Switch

global mixins

전역으로 선언되어 있는 mixin는 공통적으로 컴포넌트에서 사용하는 룰에 대해서 정의하였습니다.

global mixins는 mixins.scss에 global variables와 함께 정의되어 있습니다.

  • tau-container-text($color: $default-color, $no-wrap: true) : 컨테이너에 들어가는 텍스트에 대한 스타일

    • {color} $color : 기본값은 전역변수인 $default-color값

    • {boolean} $no-wrap: wrap 여부를 설정한다.

  • tau-content-text($font-size: 1em) : 컨텐트 영역의 텍스트에 대한 스타일

    • {size} $font-size

  • tau-title ($font-size: 1em) : 제목 텍스트에 대한 스타일

    • {size} $font-size

  • tau-content : 컨텐트 영역에 대한 스타일

  • tau-container : 컨텐트 영역에 대한 스타일

  • tau-default($color: $default-color) : 기본 상태일 때 스타일

    • {color} $color

  • tau-selected($color: $selected-color) : 선택이 되었을 때 스타일

    • {color} $color

  • tau-disabled($color: $disabled-color) : 선택할 수 없는 상태일 때 스타일

    • {color} $color

  • tau-highlighted($color: $highlighted-color) : 포커스되었을 때 스타일

    • {color} $color

  • tau-vertical-content : 수직방향의 컨텐트에 대한 스타일

  • tau-horizontal-content : 수평방향의 컨텐트에 대한 스타일

  • tau-mask($position: absolute, $bgcolor: $mask-bgcolor) : 마스크에 대한 스타일

    • $position : 기본값은 absolute

    • $bgcolor: 마스크에 대한 배경색을 설정한다. 기본값은 $mask-bgcolor

  • tau-popup : 팝업에 대한 스타일

  • tau-full($position: absolute) : full 스크린에 대한 스타일

    • $position: 기본값은 absolute

  • tau-round : border를 둥근게 하기 위한 스타일

  • tau-badge($font-size: 1em, $margin-size: $badge-margin) : 버튼, shortcut 등에 사용되는 배지에 대한 스타일

    • $font-size: 폰트 크기

    • $margin-size: 배지의 크기의 절반으로 보고 설정하면 됨.

  • tau-badge-text($size: $badge-width, $margin-size: 2px) : 배지 텍스트에 대한 스타일

    • $size: 텍스트 영역의 크기

  • tau-gradient($start-bgcolor, $stop1-bgcolor, $stop2-bgcolor, $end-bgcolor) : Mobello에서 미리 정의한 gradient이 4단계(0%, 50%, 51%, 100%) 로 정의되어 있다.

  • tau-inline-flexbox($important:'') : inline flexbox에 대한 스타일

컴포넌트 mixins

기본적으로 모든 컴포넌트는 mixin로 되어 있습니다. 사용하기 위해서는 include해서 사용할 수 있습니다.

다만, 현재 tau.scss파일에서 all.scss파일을 include하도록 지침을 내리고 있는데 all.scss파일에서 컴포넌트 mixin들을 include해서

따로 사용자는 all.scss파일만 include하면 다른 작업을 할 필요는 없습니다. 단, 컴포넌트 자체 정의되어 있는 mixin을 사용하지 않고 사용자가 직접 컴포넌트에 대한 scss파일로 대체한 경우에는

필요한 컴포넌트에 대해서 include하시면 됩니다.

Button

  • tau-button-type : 버튼 타입 테마를 추가하기 위한 mixin 입니다.

  • tau-button-state : tau-button-type와 함께 사용하는 mixin으로 상태별(selected | highlighted | disabled)에 대해 설정하는 mixin입니다.

  • tau-button-box-shadow : 버튼 영역에 shadow를 지정하기 위한 mixin입니다.

ScrollPanel

  • tau-scroller : scroller를 사용할 경우 include하면 됩니다. 현재 all.scss파일에서 include하고 있습니다.

  • tau-pulltorefresh : pulltorefresh를 사용할 경우 include하면 됩니다. 현재 all.scss파일에서 include하고 있습니다.

Switch

  • tau-switch-bar : Switch 내부에서 beforebar, afterbar에 대한 스타일을 적용하는데 사용하는 mixin입니다.
